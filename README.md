# Airbnb Listings (JavaScript & DOM)

This project displays the first 50 Airbnb listings loaded from a JSON file using JavaScript `fetch()` with `async/await`.

## Live Demo
https://navanika2211.github.io/airbnb-listings/

## Features
- AJAX data loading using fetch and async/await
- Displays listing name, description, amenities, host name & photo, price, and thumbnail
- Search bar to filter listings (creative addition)

## How to Run
Open using GitHub Pages or VS Code Live Server.
